## 👨‍💻 Tech stack

* [Nuxt/tailwindcss](https://github.com/nuxt-community/tailwindcss-module)
* [Nuxt/image](https://image.nuxtjs.org/)

🖥 Backend:
* [Nest.js](https://nestjs.com/)
* MongoDB
